[ 简介 ]
小微OA系统是一个免费开源的，简单易用，审批灵活，支持移动办公，功能实用的OA系统。
遵循Apache2开源协议发布，是为了帮助快速成长的小微企业实现信息化而诞生的。


[ 安装方法 ]

http://my.oschina.net/oaoa/blog/161945

1.ThinkPhp目录剪切到上一级目录

2.运行Install.php 或 用MySQL-Front直接导入 SQL\demo.sql 文件

3.不支持PATHINFO或REWRITE请修改conf\config.php 文件中  
   
   'URL_MODEL'=>2,  //普通模式 1，PATHINFO模式 2，REWRITE 模式 3
